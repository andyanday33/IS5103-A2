WAVE

WAVE is a useful evaluation tool for helping web developers improve their web page's accessibility, especially disabled people. WAVE also can identify content and WCAG errors, and provide help for human evaluation. It’s quite easy to test, uses can type in their URL in the WAVE’s webpage directly In addition, it provides Chrome and Firefox extensions to test your web.

There is also a series of WAVE Browser Extensions which allow you to evaluate web content for accessibility issues directly within your browser.

Colour blindness simulation

Coblis is a simulation tool for color blindness. It can accurately simulate the perceptual barriers of outstanding blindness so that developers can better improve the accessibility of web pages for color-blind users. In addition, he provides some knowledge bases and color testing tools about color blindness. Of course, there are some Q&A about color blindness included in it.
